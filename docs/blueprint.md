# **App Name**: CulinaryFlow

## Core Features:

- Unified Login System: Routes users to Admin or Worker views based on ID prefix.
- Admin Portal (Menu CMS): CRUD operations for menu items with availability toggle and live preview.
- Worker Portal (KDS): Live order feed with status updates and emergency stock controls.
- Shared State Management: Ensures 'Sold Out' status updates are reflected across both Admin and Worker views.
- Mock Data Population: Pre-populates menu items and active orders for immediate interactivity.
- Dashboard Overview: Show high-level information about the business such as revenue and active orders.

## Style Guidelines:

- Primary color: Deep burgundy (#800020) to evoke the rich flavors and cozy ambiance of a culinary environment.
- Background color: Light tan (#F5E7D1), a desaturated and brighter hue from the primary color for a warm and inviting base.
- Accent color: Muted orange (#A66321), analogous to burgundy but with less saturation for highlights and call-to-action elements.
- Body font: 'Inter', a sans-serif for clean, modern readability.
- Headline font: 'Belleza', a serif font for headers. Pairing instructions: if longer text is anticipated, choose 'Alegreya' for body text
- Lucide React icons used throughout for a consistent and modern feel.
- Shadcn UI components (Cards, Inputs, Buttons) employed for a polished and consistent design language.
